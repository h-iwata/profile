# Test Specifications

このディレクトリには、テスト仕様書（.md ファイル）を配置します。

## ファイル構造

- `navigation.test.md` - ナビゲーションテストの仕様
- その他のテスト仕様書...

## 仕様書の書き方

各仕様書は以下の形式で記述します：

```markdown
# [機能名]のテスト仕様

## テスト目的
[テストの目的を記述]

## テストケース

### ケース1: [テストケース名]
- **前提条件**: [前提条件]
- **手順**: 
  1. [手順1]
  2. [手順2]
- **期待結果**: [期待される結果]
```
